##A1标准开发平台是什么?
* SOA控件化开发平台,可自定义控件的开发平台

##A1标准开发平台有哪些功能？

* 1.表单可配置
* 2.查询列表可配置
* 3.树形可配置
* 4.框架可配置
* 5.菜单可配置
* 6.定时任务可配置
* 7.发布订阅可配置(功能完成，无界面)
* 8.数据库可配置(功能初始化完成,添加,修改暂不能同步，无界面)

##开发环境
* java v 1.7

##使用技术:
* ActiveMQ 5.13.2
* Activiti 5.22.0
* Apache Shiro 1.2.3
* Dubbo 2.8.4
* RedisDb 3.0
* MariaDB 10.1.9
* Apache FTP 1.1.1
* ZooKeeper 3.4.6

##平台目标
* 1.支持大数据
* 2.安全可靠
* 3.快速高效可配置
